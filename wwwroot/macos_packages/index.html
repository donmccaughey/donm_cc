Signed and Notarized Universal macOS Installer Packages

These standard macOS installer packages are built using pkgbuild and productbuild and are signed and notarized with my Apple developer credentials. They install universal binaries that will run on both Intel and Apple Silicon Macs.

This is a collection of command line tools that I've found useful at one time or another over the years. Some of them are widely used but excluded from macOS and Xcode due to GPL licenses.
